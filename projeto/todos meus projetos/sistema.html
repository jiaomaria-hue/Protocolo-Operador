Converta os algoritmos em código Python real dentro do seu arquivo projeto.py de cabeça.

⚡ Desafio 1.1: O Disparador de Alertas

Missão: Faça o terminal mostrar uma mensagem estática de aviso, informando que o sistema de segurança foi ligado.

A Lógica na sua cabeça:

1. Chamar a função de mostrar texto na tela.
2. Passar a mensagem: "[SISTEMA INICIALIZADO]"

Ferramenta necessária: print()

⚡ Desafio 1.2: Captura de IP do Alvo

Missão: Faça o programa perguntar um endereço IP para quem estiver usando, guarde essa resposta e depois mostre uma frase confirmando qual IP foi capturado.

A Lógica na sua cabeça:

1. Criar uma variável chamada 'alvo'.
2. Fazer o programa perguntar o IP e salvar dentro de 'alvo'.
3. Dar um print juntando o texto fixo com o valor da variável.

Ferramentas necessárias: variáveis e input()

⚡ Desafio 1.3: Bloqueador de Palavras Perigosas

Missão: Peça para o usuário digitar um comando. SE o comando for exatamente a palavra "virus", dispare um alerta de perigo vermelho na tela.

A Lógica na sua cabeça:

1. Receber o comando via input.
2. Testar a condição: SE o comando for igual a "virus", empurre a próxima linha para a direita e mostre o alerta.

Ferramentas necessárias: input(), if e operador de igualdade (==)

⚡ Desafio 1.4: O Inspetor de Portas de Rede

Missão: Crie um validador de portas. Pergunte o número de uma porta. SE for a porta 80, diga que é tráfego de internet comum. CASO CONTRÁRIO (qualquer outra porta), exiba que a porta é suspeita.

A Lógica na sua cabeça:

1. Receber o número transformando em número inteiro.
2. SE for igual a 80 -> Mostra "Porta HTTP".
3. CASO CONTRÁRIO (na mesma linha do if original) -> Mostra "Suspeita".

Ferramentas necessárias: int(input()), if e else

👑 Desafio 1.5: Triagem de Sistema de Arquivos (Malware Scan)

Missão: Peça o nome de um arquivo com o formato. SE terminar com ".exe", printe um aviso de perigo. SE terminar com ".py", printe que é um script de automação. CASO CONTRÁRIO, trate como arquivo comum.

A Lógica na sua cabeça:

1. Capturar o nome do arquivo.
2. SE terminar com ".exe" -> Alerta.
3. SENÃO SE terminar com ".py" -> Script Seguro.
4. SENÃO -> Arquivo comum.

Ferramentas necessárias: if, elif, else e a função .endswith()
