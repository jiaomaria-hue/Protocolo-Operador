## 💻 LINGUAGENS — NESSA ORDEM, SEM PULAR

1. **Agora:** Python (scripts, automação, exploits simples)
2. **Fase 2–3:** Bash (terminal Linux, aprofundar)
3. **Fase 4:** PowerShell (ambientes Windows e Active Directory)
4. **Fase 5:** C / C++ (ler código de malware, engenharia reversa)
5. **Fase 5+:** Assembly (só se for especializar em reversão)

> *Regra: Não aprende linguagem por curiosidade. Aprende quando um problema exige.*

---

## FASE 1 — DOMÍNIO DA MÁQUINA (Meses 1–3)

**Onde você sai de usuário para Operador.**

* **O que aprender:** Linux terminal (permissões, usuários, processos), Python puro (lógica, tratamento de erros) e Bash básico.
* **Fontes:** SecDay Academy (16 aulas) e Cisco (Introdução à Cibersegurança).
* **O que entregar:** Scripts de gestão de sistema, projeto `firewall_logic.py`, GitHub ativo e VirtualBox com VM criada.

---

## FASE 2 — VIGILÂNCIA DA REDE (Meses 4–6)

**Entender como os dados se movem é entender o campo de batalha.**

* **O que aprender:** Modelo OSI, TCP/IP, DNS, DHCP, HTTP/S, ARP.
* **Ferramentas:** Wireshark (análise), Nmap (mapeamento), Netcat (conexão), Scapy (pacotes).
* **O que entregar:** Port Scanner em Python e capturas de tráfego documentadas no `LOG_DE_GUERRA.md`.

---

## FASE 3 — ARTE DA OFENSIVA / RED (Meses 7–10)

**Mentalidade de invasor. Se a máquina não cair, você não para.**

* **O que aprender:** OWASP Top 10, Engenharia de exploits, CVEs, Metodologia de ataque.
* **Ferramentas:** Burp Suite, Metasploit.
* **Plataformas:** TryHackMe (caminho *Jr Pentester*) e depois HackTheBox.
* **O que entregar:** Caminho do TryHackMe completo e write-ups das máquinas no GitHub.

---

## FASE 4 — ARTE DA DEFESA / BLUE (Meses 11–13)

**Você só defende bem o que aprendeu a atacar.**

* **O que aprender:** Hardening, análise de logs, Firewalls (UFW/Iptables), SIEM (Wazuh/Splunk), Cloud Security.
* **O que entregar:** Hardening das máquinas que você invadiu na Fase 3 e configuração de SIEM para detectar seus próprios ataques anteriores.

---

## FASE 5 — CONSIDERAÇÃO DE ELITE (Meses 14–18)

**Purple Team completo: você ataca, detecta, responde e endurece.**

* **O que aprender:** Active Directory, Cloud Security avançado, Engenharia Reversa (Ghidra/radare2), MITRE ATT&CK.
* **O que entregar:** Projetos de automação de Purple Team, write-ups de simulações, certificações eJPT e CompTIA Security+.
